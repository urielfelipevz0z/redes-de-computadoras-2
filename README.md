# Redes de Computadoras 2

Este repositorio contiene la documentación completa de las prácticas de laboratorio para la materia **Redes de Computadoras 2** (CCNA: Fundamentos de Conmutación, Enrutamiento y Redes Inalámbricas) de la Universidad Michoacana de San Nicolás de Hidalgo de la Facultad de Ingeniería Eléctrica.

## Metodología de Documentación

### Framework Tecnológico
- **Pandoc Markdown**: Documentación científica y técnica
- **Eisvogel Template**: Plantilla LaTeX para PDFs profesionales
- **Mermaid**: Diagramas de red y topologías
- **GitHub**: Control de versiones y colaboración

### Estándares de Calidad
1. **Documentación completa** del proceso de aprendizaje
2. **Evidencias técnicas** con capturas y configuraciones
3. **Automatización** de tareas repetitivas
4. **Versionado** de configuraciones y scripts
5. **Reproducibilidad** de experimentos

### Convenciones de Nomenclatura
- **Prácticas**: `practica-XX-[tema-descriptivo]`
- **Configuraciones**: `[DISPOSITIVO]-[ESTADO]-[VERSION].cfg`
- **Imágenes**: `[seccion]-[descripcion]-[numero].png`
- **Scripts**: `[funcion]_[proposito].[extension]`
